# 报告生成

`report.py` — 读取四张飞书表（自选/持仓/交易/现金），生成持仓报告。

## 触发场景

- 用户说「报告」「持仓报告」「今日报告」「我今天赚了多少钱」
- 每日三个定时：11:30 盘中 / 16:30 收盘 / 20:30 晚间

## 命令

```bash
# 自动推断时间版本
python scripts/report.py

# 显式指定时间
python scripts/report.py --time 11:30               # 盘中报告
python scripts/report.py --time 16:30               # 收盘报告
python scripts/report.py --time 20:30               # 晚间报告（完整版）

# 不爬取行情（仅生成）
python scripts/report.py --no-sync

# 强制生成（忽略未更新检查）
python scripts/report.py --force
```

## 报告类型

| 时间 | 类型 | 特点 |
|------|------|------|
| 11:30 | 盘中报告 | 基金/债券只报市值，不报涨跌 |
| 16:30 | 收盘报告 | 基金净值可能未更新，只报市值 |
| 20:30 | 晚间报告 | 完整涨跌，有重试逻辑（最多 8 次×15 分钟） |

## 今日收益计算（非交易日关键规则）

**核心公式**：
```
各标的当日收益 = 市值 − 市值 / (1 + 涨幅%)
今日收益 = Σ(股票当日收益) + Σ(基金当日收益) + Σ(债券当日收益)
         = Σ 各标的 change_amount
综合涨幅 = 今日收益 / 总持仓市值 × 100%
```

**示例**（2026-06-07 非交易日）：
| 类型 | 今日市值 | 涨幅 | 当日收益 |
|------|---------|------|---------|
| 股票 | 524,868.73 | -0.45% | -2,372.59 |
| 基金 | 827,754.19 | -0.81% | -6,759.56 |
| 债券 | 81,368.99 | -0.05% | -40.70 |
| **合计** | 1,433,991.91 | | **-9,172.85** |

```
今日收益 = -9,172.85
综合涨幅 = -9,172.85 / 1,433,991.91 × 100% = -0.64%
```

**不同情况下的计算方式**：
| 情况 | 今日收益算法 |
|------|------------|
| 交易日 + 20:30 | 今日总资产 − 昨日总资产（锚点相减） |
| 非交易日（任意时间） | Σ各标的当日收益（change_amount 之和） ✅ |
| 盘中/收盘报告 | Σ各标的当日收益 |

**注意**：
- 现金余额变动**不**计入今日收益（与市场涨跌无关）
- 基金净值未更新时，change_pct=0，当日收益计为 0

## 基准文件保护（非交易日不更新）

- 路径：`财神/reports/baseline.json`
- **非交易日不应更新基准文件**
- 更新条件：`report_time == "20:30" and is_trading_day == True`
- 典型错误：非交易日执行 20:30 报告，把 date 更新为非交易日日期，导致次日锚点计算错误

**修复方法**：在 `generate_report()` 调用时传入 `is_trading_day` 参数，并在更新基准前判断：

```python
# 只在交易日更新基准
if report_time == "20:30" and is_trading_day:
    # 更新 baseline.json
```

## 20:30 重试逻辑

- 触发条件：场外基金或债券在自选表中的"更新日期"不是今日
- 重试间隔：15 分钟一次
- 最长等待：22:30 截止（8 次重试）
- 超过截止：无论是否全部更新，直接生成报告

## 报告输出规则

- 报告**不得截断**，完整展示给用户
- 不使用 `head`/`tail`/`grep` 等过滤输出
- 非交易日报告底部标注：`（非交易日，估值参照最近交易日涨跌）`

## 图表生成

- 股票饼图：`scripts/render_charts.py render_stock_pie`，输出 `财神/charts/stock_pie_*.png`
- 基金双圈图：`scripts/render_charts.py render_fund_donut`，输出 `财神/charts/fund_donut_*.png`
- 生成后通过 `message` tool 发送给皮迪克

## 本功能常见错误

| 错误 | 原因 | 恢复 |
|------|------|------|
| 基准文件被非交易日覆盖 | 报告未判断 is_trading_day | 已修复：`report.py` 加 `is_trading_day` 条件 |
| 今日收益计算错误（非交易日） | 锚点相减失效 | 非交易日改用 Σ各标的当日收益 |
